# 🏏 IPL Analytics Dashboard

An interactive analytics dashboard built with Python & Streamlit,
analyzing IPL data from 2008 to 2024.


## 📊 Features
- Most successful teams by total wins
- Top 10 run scorers and wicket takers
- Toss impact analysis
- Best venues by average score
- Season-wise run trend (2008–2024)
- Sidebar filter by season
- Team vs Team analysis

## 🛠 Tech Stack
Python · Pandas · Matplotlib · Seaborn · Streamlit

## 🚀 Run Locally
git clone https://github.com/YOUR_USERNAME/ipl-analytics-dashboard.git
cd IPL-Analytics-Dashboard
pip install -r requirements.txt
streamlit run app.py

## 📦 Dataset
IPL Complete Dataset (2008–2024) from Kaggle
